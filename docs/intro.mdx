<Admonition type="info" icon="ðŸ’¾" title="Custom deployments...">
  The [`docker`](installation/docker/intro) and [`ansible`](installation/ansible/intro) deployments utilize a **custom installation**. This setup includes specific extensions for handling spatial data and for implementing the [GeoDCAT-AP](https://github.com/SEMICeu/GeoDCAT-AP) and [INSPIRE](https://github.com/INSPIRE-MIF/technical-guidelines) metadata [profiles](https://en.wikipedia.org/wiki/Geospatial_metadata). These profiles are standards for the representation of geospatial metadata. 
</Admonition>
