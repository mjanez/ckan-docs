<Admonition type="info" icon="💾" title="Despliegues personalizados...">
  Los despliegues [`docker`](instalacion/docker/intro) y [`ansible`](instalacion/ansible/intro) utilizan una **instalación personalizada**. Esta configuración incluye extensiones específicas para manejar datos espaciales e implementar los [perfiles](https://en.wikipedia.org/wiki/Geospatial_metadata) de metadatos [GeoDCAT-AP](https://github.com/SEMICeu/GeoDCAT-AP) e [INSPIRE](https://github.com/INSPIRE-MIF/technical-guidelines). Estos perfiles son estándares para la representación de metadatos geoespaciales. 
</Admonition>
